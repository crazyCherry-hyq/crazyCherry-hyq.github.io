博客笔记 - 杨天真vol.03 拥有真正的能量 讨论如何成为自己

在这个快速变化的时代，我们每个人都在寻找成长、自我实现以及稳定的方向。无论是个人发展、创业选择，还是内容创作，如何在不确定中找到确定性，是许多人关心的问题。希望这篇文章能为你提供一些思考和启发。

1. 成长：如何真正提升自己？

成长的本质是不断尝试、不断修正，并从错误中学习。每个人的成长都离不开模仿和学习，但真正的进步来自于深入思考和实践的结合。

如何将已有的优势转化为真正的价值？

  • 认清自己真正想要的，而不是盲目跟随他人的标准。
  • 设定符合自己价值观的目标，并一步步落地执行。
  • 不断探索，让自己的人生变得更有意义，而不仅仅是“成功”。

面对“不配感”，如何增强自信？

  • 每个人都会经历自我怀疑，但真正的成长来自接受自己的独特性。
  • 认可自己的优点，避免总是把目光放在自己不足的地方。
  • 关注自己已经取得的进步，而不是只看到未来的不确定性。

2. 创业与人生选择：如何做出正确决策？

创业的第一步：如何找到正确的起点？

  • 许多人想创业，但最难的是迈出第一步，比如离开舒适圈。
  • 创业的关键是找到真实的市场需求，而不是仅凭想象。
  • 通过市场调研和规划，找到痛点，创造价值。

成长的关键：如何整合资源与能力？

  • 个人能力固然重要，但更关键的是如何连接外部资源。
  • 多接触高纬度的信息，学习新趋势，保持与世界的连接。
  • 通过实践与理论的结合，让自己始终处于成长状态。

如何应对年龄焦虑？

  • 29岁的焦虑，往往来自对未来的不确定性，但真正重要的是当下的行动。
  • 生命无常，但只要专注于那些确定且有价值的事情，就能找到方向。

3. 心态与环境：如何让自己保持清醒？

如何在外界影响下坚持自己的方向？

  • 环境和舆论对个人发展有深远影响，学会甄别信息，避免被误导。
  • 面对质疑和批评，保持清醒，不被短期情绪左右。

高标准与高要求：如何在成长中平衡压力？

  • 设定高标准是进步的关键，但如果压力过大，也会影响长期成长。
  • 学会在高要求中找到平衡，让成长成为一种乐趣，而不是负担。

规律与变化：如何在不确定中找到确定性？

  • 变化无法避免，但我们可以建立规律，让自己更稳定。
  • 养成阅读、思考、学习的习惯，提升系统性思维能力，减少焦虑。

4. 内容创作：如何用创造力赋能自己？

内容创作如何帮助个人成长？

  • 内容创作是一个沉浸式学习过程，从构思到执行，每一步都是提升自己的机会。
  • 通过不断优化拍摄、剪辑、表达方式，形成自己的风格。

规划未来：如何找到正确的商业方向？

  • 以真实的市场需求为核心，建立清晰的商业框架，而不是被外界变量牵着走。
  • 通过持续行动积累经验，让自己掌控主动权，而不是成为环境的“变量”。

关于闭包

发表于 2022-04-18

闭包的描述

源自MDN：一个函数和对其周围状态（lexical environment，词法环境）的引用捆绑在一起（或者说函数被引用包围），这样的组合就是闭包（closure）。也就是说，闭包让你可以在一个内层函数中访问到其外层函数的作用域。在 JavaScript 中，每当创建一个函数，闭包就会在函数创建的同时被创建出来

源自《javascript高级程序设计》：闭包是指有权访问另一个函数作用域的变量的函数

闭包的组成

闭包是由函数以及声明该函数的词法环境组合而成的

闭包的意义

1：延长变量的生命周期
2：创建私有环境

使用闭包需要注意的点

1：容易导致内存泄漏

程序的运行需要内存。对于持续运行的服务进程，必须及时释放不再用到的内存，否则占用越来越高，轻则影响系统性能，重则导致进程崩溃。不再用到的内存，没有及时释放，就叫做内存泄漏

2：注意闭包的this指向

关于 var、let 和 const 三个关键字的区别

发表于 2022-04-07 更新于 2022-04-20 分类于 Record

es5的var：

  1. 定义的变量是全局的
  2. 如果在变量声明前取值会报undefined （存在变量提升）
  3. 声明不会形成作用域
  4. 可以同时声明一样变量名

es6的let：

  1. 定义的变量是在词法作用域范围内有用
  2. 如果在变量声明前取值会报错 （不存在变量提升）
  3. 声明会形成作用域
  4. 不可以同时声明一样变量名

es6的const：

  1. 定义的变量是常量，不可以修改和重新赋值，不能使用null做占位符
  2. 如果在变量声明前取值会报错 （不存在变量提升）
  3. 声明会形成作用域
  4. 不可以同时声明一样变量名

const定义变量的注意点

①：一定要赋初始值
②：一般常量使用大写（潜规则）
③：常量的值不能修改
④：块级作用域
⑤：对于数组和对象的元素修改，不算做对常量的修改，不会报错
